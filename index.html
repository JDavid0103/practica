> Estructura básica HTML

Aquí muestro un ejemplo de un archivo HTML básico. Como se puede ver, está estructurado por etiquetas. Comenzamos con la etiqueta <html>, la cual envuelve a todos los demás elementos, y dentro de esta tenemos tanto la cabecera del documento <head> como el cuerpo del documento <body>.

Dentro del <head> tenemos otras etiquetas como las de metadatos <meta>, las cuales se usan para dar un poco de contexto al documento. Existen varias, como las de descripción, autor, viewport...
Por otro lado, tenemos el <body> con todos los elementos de nuestra web.

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Documento HTML</title>
</head>
<body>
  <h1>Hola!</h1>
</body>
</html>

> Imágenes

La etiqueta <img> nos permite introducir imágenes desde una URL o ruta a nuestra página web, pasándosela por el atributo "src". Además, tenemos otros atributos que podemos modificar, como la altura ("height") o la anchura ("width"), o incluso agregar una descripción por si la imagen no carga con el atributo "alt".

<img src="https://www.epitech-it.es/wp-content/uploads/2024/02/pexels-pixabay-270404.jpg" height="300px" alt="Imagen de HTML">

> Video

La etiqueta <video> funciona de manera parecida a la etiqueta de imagen. Nos permite cargar vídeos en nuestra web desde una URL o ruta mediante el atributo "src". También podemos mostrar los controles del reproductor de vídeo con "controls", hacer que el vídeo se reproduzca en bucle con "loop", que se reproduzca automáticamente con "autoplay"... Igualmente, también tenemos los atributos de altura ("height") o anchura ("width") si los necesitamos.

<video src="src/video.mp4" controls></video>

> Audio

La etiqueta <audio> funciona exactamente igual que la etiqueta de vídeo. Nos permite cargar audios en nuestra web desde una URL o ruta mediante el atributo "src". También podemos mostrar los controles del reproductor de audio con "controls".

<audio src="src/audio.mp3" controls></audio>

> Lista desordenada

La etiqueta <ul> permite crear listas desordenadas. Cada elemento de la lista se agrega con la etiqueta <li>.

<ul> <li>Elemento 1</li> <li>Elemento 2</li> <li>Elemento 3</li> </ul>
  • Elemento 1
  • Elemento 2
  • Elemento 3

> Lista ordenada

La etiqueta <ol> se utiliza para crear listas ordenadas, es decir, listas numeradas. Igualmente, cada elemento de la lista se agrega con la etiqueta <li>.

<ol> <li>Primer elemento</li> <li>Segundo elemento</li> <li>Tercer elemento</li> </ol>
  1. Primer elemento
  2. Segundo elemento
  3. Tercer elemento

> Enlace

La etiqueta <a> se utiliza para crear enlaces a otras páginas o rutas dentro de la misma página. El atributo principal es href, que indica la URL de destino. También se puede usar target="_blank" para abrir el enlace en una nueva pestaña.

<a href="https://www.google.es" target="_blank">Ir a Google</a>
Ir a Google

> Canvas

La etiqueta <canvas> se utiliza para dibujar gráficos mediante JavaScript. Por sí sola, no muestra nada; se necesita código para dibujar dentro de ella.

<canvas id="miCanvas"></canvas>
