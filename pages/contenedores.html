> Etiqueta body

La etiqueta <body> contiene todo el contenido visible de una página web: textos, imágenes, enlaces, formularios, secciones, etc. Solo puede haber un <body> por documento HTML y se muestra dentro de la ventana del navegador.

> Etiqueta header

La etiqueta <header> define la cabecera de una página o de una sección. Normalmente contiene el logotipo, el título, el menú de navegación o información introductoria. Puede usarse más de una vez, por ejemplo, dentro de un <article> o <section>.

> Etiqueta main

La etiqueta <main> representa el contenido principal de una página. Debe ser única y contener la información más relevante o específica del documento, excluyendo encabezados, menús, barras laterales o pies de página.

> Etiqueta footer

La etiqueta <footer> define el pie de página de un documento o sección. Suele incluir información de contacto, derechos de autor, enlaces secundarios o créditos. Puede aparecer tanto al final de la página como en artículos o secciones individuales.

> Etiqueta div

La etiqueta <div> es un contenedor genérico que se utiliza para agrupar elementos y aplicarles estilos o comportamientos comunes mediante CSS o JavaScript. No tiene un significado semántico por sí misma.
