Formularios

> Etiqueta form

En HTML podemos crear formularios, para ello tenemos que usar la etiqueta <form>. Dentro de ella se incluyen los campos y controles que permiten al usuario introducir datos.

> Etiqueta label

La etiqueta <label> se utiliza para definir una etiqueta descriptiva de un campo de formulario. Está asociada a un campo mediante el atributo for, que coincide con el id del elemento correspondiente.

> Etiqueta input

La etiqueta <input> crea un campo de entrada donde el usuario puede escribir o seleccionar información. Su comportamiento depende del valor del atributo type (por ejemplo: text, email, password, radio, checkbox, file, color, date etc.), ademas es importante ponerle una id para poder recuperar el elemento.

> Etiqueta fieldset

La etiqueta <fieldset> agrupa visual y semánticamente varios controles de un formulario. Suele utilizarse para organizar los datos en secciones.

> Etiqueta legend

La etiqueta <legend> se usa dentro de un <fieldset> y define un título o leyenda para ese grupo de campos, ayudando a identificar su contenido.

> Etiqueta div

La etiqueta <div> sirve para agrupar contenido y aplicar estilos o estructura dentro de un formulario o cualquier parte del documento HTML.

> Etiqueta select

La etiqueta <select> permite crear un menú desplegable en el que el usuario puede elegir una opción de una lista predefinida.

> Etiqueta option

La etiqueta <option> define cada una de las opciones dentro de un menú desplegable <select>. Cada opción tiene un atributo value que indica el valor que se enviará al formulario.

> Etiqueta textarea

La etiqueta <textarea> crea un campo de texto multilínea. Se puede ajustar su tamaño mediante el atributo rows, que define el número de líneas visibles, y se puede mostrar un texto orientativo con el atributo placeholder, por ejemplo: <textarea rows="5" placeholder="Introduzca su opinión"></textarea>.
